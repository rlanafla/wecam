# WeCam 프론트엔드 실행 가이드 (Docker)

디자이너/기획자도 로컬 환경에서 WeCam 프론트를 바로 확인할 수 있도록 Docker 기반 실행 환경을 제공합니다.  
Node.js, npm, Next.js 설치 없이 **Docker만 설치**되어 있으면 됩니다.

---

## ✅ 사전 준비

- [Docker Desktop 설치](https://www.docker.com/products/docker-desktop)
